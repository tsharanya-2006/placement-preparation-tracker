# Placement Preparation Tracker

## Overview
Placement Preparation Tracker is a Python-based console application that helps students organize and track their placement preparation.

## Features
- Add DSA problems
- Add projects
- View progress
- Mark goals as completed
- Delete DSA problems
- Delete projects
- Search DSA problems
- Search projects
- Edit DSA problems
- Edit projects
- View statistics
- Save data using file handling

## Technologies Used
- Python
- File Handling

## How to Run
1. Download the project.
2. Open it in VS Code.
3. Run `placement_tracker.py`.
